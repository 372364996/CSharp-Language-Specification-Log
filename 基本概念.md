## 应用程序入口点方法Main

具有下列签名之一：

```
static void Main(){...}
static void Main(string[] args){...}
static int Main(){...}
static int Main(string[] args){...}
```

1. string\[\] args参数：
   1. 由执行环境创建并传递一个包含**命令行实参**的string\[\]实参，命令行实参是在启动应用程序时指定的。
   2. string\[\]参数永远不能为null，没有指定则长度为0；
